Nuevos Elementos semánticos

  • Article Define un post, comentario, articulo, otros

  • Aside Define un contenido no relacionado al documento

  • Figcaption Define el a caption para un elemento figure

  • Figure Especifica contenido como ilustraciones, diagramas, etc

  • Footer Especifica el pie del documento o seccion

  • Header Especifica el encabezado del documento o seccion

  • Mark Define texto remarcado

  • Nav Define enlaces de navegacion

  • Section Define una seccion del documento

  • Time Define tiempo/fecha
